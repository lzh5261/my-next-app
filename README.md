## Getting Started

### Development
```bash
git clone https://github.com/lzh5261/my-next-app.git
cd my-next-app
yarn
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Deploy
```bash
git clone https://github.com/lzh5261/my-next-app.git
cd my-next-app
yarn
yarn build
```

Open [http://vercel.com](http://vercel.com) to deploy.
